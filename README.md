# 跨年倒计时时钟 🎉

一个个性化的Python跨年倒计时时钟程序，具有精美的模拟时钟界面和跨年特效。

## 功能特点

- ⏰ **模拟时钟界面**：传统的指针式时钟，带有精美的表盘设计
- 🎨 **个性化设计**：金色主题，黑色背景，视觉效果优雅
- ⏱️ **实时时间显示**：数字时间显示和指针时钟同步
- 🎆 **跨年倒计时**：显示距离新年的精确倒计时
- 🎊 **烟花特效**：跨年前1分钟和跨年后自动播放烟花动画
- 📅 **日期显示**：显示当前日期

## 运行要求

- Python 3.6 或更高版本
- tkinter（Python标准库，通常已包含）

## 使用方法

1. 确保已安装Python 3.6+

2. 运行程序：
```bash
python new_year_countdown.py
```

或者：

```bash
python3 new_year_countdown.py
```

## 程序说明

- 程序会自动显示当前时间
- 时钟指针会实时更新（时针、分针、秒针）
- 显示距离新年的倒计时
- 跨年前1分钟开始播放烟花特效
- 跨年后显示新年祝福并持续烟花效果

## 界面说明

- **顶部**：程序标题
- **中央**：模拟时钟表盘，带有12小时刻度和60分钟刻度
- **底部**：数字时间显示、日期和倒计时信息

## 技术特点

- 使用tkinter创建图形界面
- 实时更新机制（每100毫秒刷新一次）
- 物理引擎模拟烟花粒子效果
- 数学计算精确绘制时钟指针

## 自定义

你可以通过修改代码中的以下参数来自定义程序：

- `self.radius`：时钟半径
- `self.center_x, self.center_y`：时钟中心位置
- 颜色值（如 `#FFD700` 金色，`#0a0a0a` 背景色）
- 字体大小和样式
